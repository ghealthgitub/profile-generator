## ğŸ› Troubleshooting

### "Could not extract data from URL"

**Causes:**
- Website blocks scraping
- Invalid URL
- Page requires login
- Network issue

**Solutions:**
1. Check URL is correct
2. Try different page for same doctor
3. Check internet connection
4. Try again in a few minutes

---

### "No procedures matched"

**Causes:**
- Doctor is very specialized (rare specialty)
- Webpage has minimal information
- Doctor does general practice

**Solutions:**
1. Try more detailed source page
2. Manually add procedures in prompt
3. Use general specialty descriptions

---

### "Document won't download"

**Causes:**
- Browser popup blocker
- Download folder permissions
- Browser cache issues

**Solutions:**
1. Allow popups for this site
2. Check download folder exists
3. Try different browser
4. Clear browser cache

---

### "Prompt too long for Claude"

**Rare, but if happens:**
1. System automatically truncates text
2. Edit prompt to remove excess
3. Focus on most relevant sections
